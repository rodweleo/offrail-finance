## Performance Considerations

```
Initial Render
    в”ңв”Җ useAccount() вҶ’ Check if connected
    в”ңв”Җ useConnect() вҶ’ Get available connectors
    в”ңв”Җ useEffect() вҶ’ Check localStorage
    в””в”Җ Render UI

On Connection
    в”ңв”Җ save to localStorage (1ms)
    в”ңв”Җ wagmi connect() (async)
    в”ңв”Җ useAccount() updates
    в””в”Җ Re-render (fast)

Storage Lookup
    в”ңв”Җ Synchronous read
    в”ңв”Җ ~0.1ms on typical device
    в””в”Җ Runs once on mount
```
